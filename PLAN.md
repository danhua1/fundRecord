# 项目规划文档

## 项目背景

以基金投资研究为核心场景，利用 A 股公开数据 API，构建从「大盘环境判断 → 板块选择 → 资金验证 → 基金筛选」的完整决策链路。

## API 清单

### 基金数据（fund_advisor.py）

| API | 端点 | 说明 |
|-----|------|------|
| 全量基金列表 | `fund.eastmoney.com/js/fundcode_search.js` | 27000+ 只基金 |
| 基金排名 | `fund.eastmoney.com/data/rankhandler.aspx` | 按类型/时段排名 |
| 净值历史 | `api.fund.eastmoney.com/f10/lsjz` | 日级净值，支持分页 |
| 持仓明细 | `fundf10.eastmoney.com/FundArchivesDatas.aspx` | 前十大重仓股 |
| 实时估值 | `fundgz.1234567.com.cn/js/{code}.js` | 当日估算净值 |

### 行情数据（a-stock-data skill）

| 层级 | 数据 | 接口 |
|------|------|------|
| 行情层 | K线/盘口/PE/PB/指数/ETF | mootdx + 腾讯 + 百度 |
| 研报层 | 研报列表/PDF/评级/一致预期 | 东财 + 同花顺 + iwencai |
| 信号层 | 热点题材/北向资金/板块归属/龙虎榜/解禁 | 同花顺 + 东财 |
| 资金面 | 融资融券/大宗交易/股东户数/分红/资金流 | 东财 datacenter |
| 新闻层 | 个股新闻/全球资讯 | 东财 |
| 基础数据 | 财务/F10/三表 | mootdx + 东财 + 新浪 |
| 公告层 | 公告检索/下载 | 巨潮 |

## 评分模型

```
score = 0.30 × 近1年收益(归一化)
      + 0.20 × 近3年收益(归一化)
      + 0.15 × 近6月收益(归一化)
      + 0.10 × 规模适中得分
      - 0.15 × 费率惩罚
      - 0.10 × 规模惩罚
```

- 费率惩罚：管理费 1.5% 扣 0.5 分
- 规模惩罚：<5000 万（清盘风险）或 >80 亿（灵活性下降）
