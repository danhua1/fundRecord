# 更新日志

## 2026-06-23 - V2.0 增强版

### 🎯 核心改进

#### 1. 修复基金列表正则表达式问题
- **问题**：原正则表达式长度限制过严（`{1,20}`, `{1,50}`等），导致只能匹配2只基金
- **修复**：优化为 `(\d{6})` 和 `([^"]+)`，成功拉取 26,956 只基金
- **影响**：所有版本的基金推荐从空数据变为正常显示

#### 2. 新增板块头部主动管理基金推荐 ⭐️
在V3成长版和V4行业轮动版中，除了提供板块ETF外，新增"💎 各板块头部主动管理基金"板块：

**V3 成长版新增**：
- 消费板块 TOP 3 主动基金
- 科技板块 TOP 3 主动基金  
- 医药板块 TOP 3 主动基金

**V4 行业轮动版新增**：
- 根据实时行业涨幅，为 TOP 5 强势行业各推荐 3 只头部主动基金
- 动态匹配：半导体、新能源、人工智能、医药、消费等

#### 3. 新增功能函数
```python
def find_sector_top_funds(sector: str, rank_data: list[dict], top_n: int = 5) -> list[dict]
```
- 根据板块名称从主动管理基金中筛选头部基金
- 通过基金名称关键词匹配 + 多维度评分排序
- 自动过滤ETF/指数基金，只保留主动管理基金（混合型/股票型）

## 使用建议

### 对于不同投资风格：

**保守型投资者**：
- 关注 V1 稳健版的债券基金配置
- 忽略主动管理基金板块

**成长型投资者**：
- 重点查看 V3 成长版的 "💎 各板块头部主动管理基金"
- 对比 ETF 和主动基金的历史收益

**交易型投资者**：
- 使用 V4 行业轮动版追踪强势板块
- 参考主动基金作为 ETF 的补充选择

**长期定投者**：
- V5 指数版提供核心宽基 + 行业卫星配置
- 低费率、纯被动策略
